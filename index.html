The wiki-data matcher searches for an exact match using all the fields you specify in your search. The "Company Name" query matches against:

  • Legal Name
  • Previous Name(s)
  • Trades As Name(s)

If the exact search fails to find a match, it searches again looking for matches against each field, along with the Company Name field, and then finds the common results.

During this second set of searches, all the following fields of a wiki-data record are examined:

  • Legal Name
  • Previous Name(s)
  • Trades As Name(s)
  • Company Website
  • Operational PO Box
  • Operational Floor
  • Operational Building
  • Operational Street 1
  • Operational Street 2
  • Operational Street 3
  • Operational Postcode

The following fields are NOT examined individually, as they would return too many results:

  • Trading Status
  • Registered Country
  • Operational City
  • Operational State
  • Operational Country
